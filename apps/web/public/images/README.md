# Images Directory

Place general images here such as:
- Logo files
- Background images
- Icon assets
- General graphics

## Recommended Formats
- PNG for transparency
- JPG for photos
- SVG for icons and logos
- WebP for optimized web delivery

## Example Files
- `logo.svg` - Main logo
- `logo-dark.svg` - Dark mode logo
- `hero-background.jpg` - Hero section background
- `workshop.jpg` - Workshop/about section image
