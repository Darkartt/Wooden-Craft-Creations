# Projects Directory

Place project images here for the gallery.

## Naming Convention
Use descriptive names:
- `oak-dining-table-1.jpg`
- `oak-dining-table-2.jpg`
- `wall-shelves-pine.jpg`
- `garden-bench-cedar.jpg`

## Image Guidelines
- **Resolution:** Minimum 1200px width for best quality
- **Aspect Ratio:** 4:3 or 16:9 recommended
- **File Size:** Optimize to under 500KB per image
- **Format:** JPG for photos, PNG if transparency needed

## Organization
You can create subdirectories by category:
- `furniture/`
- `decorations/`
- `outdoor/`
- `custom/`
